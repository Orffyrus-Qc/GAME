# GAME

Collection of game server tools, auto-installers, mods, and utilities.

## Projects

- [7DaysToDie_Server_AutoInstaller](./7DaysToDie_Server_AutoInstaller) — Automated installer batch script for 7 Days To Die game server and mods

Each subfolder is a self-contained project.

## Stationeers (IC10 Scripts & Server Tools)

The `STATIONEERS/` subdirectory contains a large curated collection of Stationeers IC10 automation scripts, logic reference databases, markup support, and dedicated server installation utilities.

See [STATIONEERS/README.md](./STATIONEERS/README.md) for the full index of scripts (centrifuge automation, atmosphere control, power, lighting, hydroponics patrols, terraforming mod, server installer, and more).
